Adds to twitter cards meta tags to head with logic based on post formats

== Description ==
This plugins creates twitter meta tags automatically, it will create different types of tags based on the post format set for the post.

== Installation ==
1. Upload the entire `lh-twitter-meta-tags` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
